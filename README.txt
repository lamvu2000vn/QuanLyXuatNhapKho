1. Yêu cầu
- Cài đặt WampServer
- PHP phiên bản từ 7.3 trở lên

2. Hướng dẫn
- Copy thư mục "src" và paste vào thư mục "C:\wamp64\www"
- Khởi chạy WampServer
- Mở trình duyệt, truy cập "localhost/phpmyadmin" đăng nhập với tài khoản "root" và mật khẩu để trống
- Tạo CSDL mới đặt tên "warehouse", chọn bảng mã "utf8_unicode_ci"
- import file "warehouse.sql" vào CSDL mới tạo
- Truy cập link "localhost/src/public/login"
- Đăng nhập với tài khoản:
	+ Tài khoản: tien@gmail.com
	+ Mật khẩu: 12345678910
- click chọn "Remember me"

* Do đây là một framework khá khó học, cộng đồng hỗ trợ không lớn và khoảng thời gian thực hiện
không nhiều nên ứng dụng còn khá nhiều lỗi, chưa đúng quy trình, giao diện còn đơn giản
và không đầy đủ các chức năng.
